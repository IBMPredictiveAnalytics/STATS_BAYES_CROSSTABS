STATS BAYES CROSSTABS Extension Command

Calculate Bayesian Test of Independence between Two Factors

STATS BAYES CROSSTABS ROWS = row variable*
COLUMNS = column variable*
SAMPLETYPE = POISSON** or JOINTMULTI or INDEPMULTI or HYPERGEOMETRIC
FIXEDMARGIN = ROWS or COLUMNS

/OPTIONS DISPLAYTABLE = YES** or NO
POSTERIOR = YES** or NO
PRIORCONCENTRATION = number
ITERATIONS = number

/SAVE WORKSPACE = CLEAR** or RETAIN
MODELFILE = "file specification"

* Required
** Default

STATS BAYES CROSSTABS /HELP prints this information and does nothing else.

STATS BAYES CROSSTABS ROWS=x COLUMNS=y
/OPTIONS PRIORCONCENTRATION =.501

Split files is not supported by this procedure. Statistics case weights are supported. Cell counts are rounded to integers after accumulation, which matters only if there are factional weights. Cases with missing values are omitted from the analysis.

The procedure calculates a crosstabulation of two factors and provides the Bayes factor for the independence assumption.

ROWS and COLUMNS keywords specify the variables forming the table. Both must have a categorical measurement level.

SAMPLETYPE specifies the sampling scheme underlying the table.

One of four sampling schemes can be specified.
  • POISSON specifies that the cell counts and grand total are random with counts having a Poisson distribution.
  • JOINTMULTI specifies random cell counts but a fixed grand total.
  • INDEPMULTI specifies that either the row or column totals are fixed, and the cell counts have a multinomial distribution within each row or column. With this scheme you must also specify which dimension is fixed.
  • HYPERGEOMETRIC specifies that both row and column margins are fixed. This can only be used with 2 x 2 tables.

For INDEPMULTI, FIXEDMARGIN must be used to specify the fixed dimension.

OPTIONS

DISPLAYTABLE specifies whether the crosstab table should be displayed.

POSTERIOR specifies that the posterior distributions of the cells be displayed.

PRIORCONCENTRATION specifies the parameter for the prior distribution. The default value is 1, which is uniform. The value must be positive. If the Bayes factor cannot be computed, slightly perturbing the value may succeed.

ITERATIONS specifies the number of iterations (samples) to use for the posterior distribution calculations. The default value is 1000.

SAVE

WORKSPACE specifies whether the workspace should be cleared or retained after the procedure completes. By default, it is cleared.

MODELFILE specifies a file name to which the model information will be written as an R workspace for later use. No file is written if this keyword is not used.
