Bayesian Crosstabs Independence Test

The procedure calculates a crosstabulation of two factors and provides the Bayes factor for the independence assumption.

The Bayes factor is the relative evidence in the data for one model over another. In this procedure it is the odds that the factors are not independent.

Split files is not supported by this procedure. Statistics case weights are supported. Cell counts are rounded to integers after accumulation. Cases with missing values are omitted from the analysis.

Rows and Columns Specify the variables to be tabulated.

Sampling Plan Four sampling schemes can be specified.

  • Poisson specifies that the cell counts and grand total are random with counts having a Poisson distribution.
  • Joint multinomial specifies random cell counts but a fixed grand total.
  • Independent multinomial specifies that either the row or column totals are fixed, and the cell counts have a multinomial distribution within each row or column. With this scheme you must also specify which dimension is fixed.
  • Hypergeometric specifies that both row and column margins are fixed. This can only be used with 2 x 2 tables.

Options

Display the crosstabulation table Check this box to display the table.

Display posterior distribution Check this box to display the posterior distribution of the cell counts

Number of samples Specify the number of iterations (samples) to be drawn in calculating the posterior distribution if it is to be displayed.

Prior Concentration Index This indexes the expected deviation from the null hypothesis under the alternative of nonindependence. The default value of 1 indicates very little prior information.

Workspace Action Choose whether to clear the workspace at the end of the procedure or retain the contents for further use in an R program in this session.

Save Model Workspace Enter a file specification to save the model workspace to a file for future use.

Additional Features

This dialog generates syntax for the STATS BAYES CROSSTABS extension command. To display syntax help for this command, run the following syntax:

STATS BAYES CROSSTABS /HELP.

In Statistics version 23 or later, you can place the cursor within the command in the Syntax Editor and press F1.
